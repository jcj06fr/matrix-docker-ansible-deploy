
Le dossier courant (installation) contient les docs necessaires
pour installer et configurer la machine maitre ansible et la machine
esclave/serveur matrix-synapse.

Nomenclature des dossiers
- prefixer par un numero suivant l'ordre des phases de l'installation.

Editeur de fichier sur les ordinateurs
- dans les docs, lorsqu'on veut editer un fichier, on faire reference a
  la commande vi mais vous pouvez utiliser n'importe quel autre editeur
  en mode ligne, comme par exemple nano
