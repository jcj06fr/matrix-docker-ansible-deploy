
Le but de ce projet est d'installer un serveur ansible-matrix-synapse
a la facon de Slavi Pantaleev 
  cf https://github.com/spantaleev/matrix-docker-ansible-deploy
et utilisable pour les besoins du reseau Solaris.

Le dossier courant (solaris) contient toutes les specificites Solaris,
principalement des docs en francais.

Tous les autres dossiers situes au niveau du dessus (.github, collections, ...)
sont issus du depot git de Pantaleev et doivent etre mis a jour regulierement
et avec precautions.

Nomenclature des docs :
- ecrire en francais ascii 7 bits (pas d'accent ni caracteres speciaux)
- ne pas depasser 80 caracteres par ligne
- stocker au format Windows (sous linux transformer les fichiers
  de doc par la commande unix2dos fichier)
- utiliser le caractere - comme separateur dans le nom de fichier
- utiliser l'extension .txt
- a chaque fois que la doc solaris decoule d'une doc Pantaleev, citer
  la reference de cette doc et plus de precision si besoin
